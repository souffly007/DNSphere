app crée un VPN local sur votre appareil qui intercepte toutes les requêtes DNS. Quand une app essaie de contacter un domaine publicitaire (ex: ads.google.com), DNSphere bloque la requête avant qu'elle n'aboutisse.

Résultat : moins de pubs, moins de tracking, navigation plus rapide !

━━━ AVANTAGES ━━━

✓ Protège TOUTES les applications (pas juste le navigateur)
✓ Aucun serveur externe - 100% local
✓ Aucune donnée collectée
✓ Pas de root nécessaire
✓ Gratuit, sans pub, sans abonnement
✓ Léger (~5 MB)
✓ Batterie préservée
